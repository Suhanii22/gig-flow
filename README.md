# GigFlow

GigFlow is a MERN stack project connecting freelancers and clients. It features gig creation, bidding, and real-time notifications.  

## Features
- Client & Freelancer roles
- Create and manage gigs
- Place and accept bids
- Real-time notifications when a freelancer is hired
- Secure login with JWT and HTTP-only cookies

## Tech Stack
- Frontend: React, Axios, Socket.IO-client
- Backend: Node.js, Express, MongoDB, Socket.IO

## Setup
1. Clone the repo: `git clone https://github.com/<your-username>/GigFlow.git`
2. Install dependencies in `backend` and `frontend`
3. Add `.env` with necessary keys
4. Run backend (`node server.js or nodemon server.js`) and frontend (`npm run dev`)
